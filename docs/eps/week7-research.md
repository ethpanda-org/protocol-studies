# 第 12 讲 | 学习小组韦尔克尔树

第 12 讲是一场专注于 Verkle Tree 的研究讲座，Verkle Tree 是当前研究和开发的一个重要领域，旨在取代 MPT 并实现无状态性。

在 StreamEth 或 Youtube 上观看 [Guillaume](https://twitter.com/gballet/)、[Ignacio](https://twitter.com/ignaciohagopian) 和 [Josh](https://twitter.com/rudolf6_) 的演讲。幻灯片 [可在此处获取](https://github.com/eth-protocol-fellows/protocol-studies/blob/main/docs/eps/presentations/week7-research.pdf)。
