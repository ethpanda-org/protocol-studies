# Hive测试

> ：警告：本文是一个[存根](https://en.wikipedia.org/wiki/Wikipedia:Stub)，通过[贡献](/contributing.md) 和扩展它来帮助维基。

[Hive](https://github.com/ethereum/hive) 是一种端到端测试工具，使模拟器能够在具有不同测试场景(模拟)的单个网络中[启动各种客户端](https://github.com/ethereum/hive/blob/master/docs/clients.md)。这是一种更简单的方法，可以在相同的参数下同时测试所有客户端以确保没有失败。这些测试还具有非常具体的边缘情况，可以针对任何可能的边缘情况和模拟/场景进行修改。由于这种简单性，Hive 在测试即将到来的分叉和网络升级方面发挥着关键作用。 

如果您好奇并想运行一些自己的测试和模拟，请[安装 Hive](https://github.com/ethereum/hive/blob/master/docs/commandline.md)！
