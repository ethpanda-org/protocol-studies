# 共识层规范 

> ：警告：本文是一个[存根](https://en.wikipedia.org/wiki/Wikipedia:Stub)，通过[贡献](/contributing.md) 和扩展它来帮助维基。

以太坊网络最初是作为工作量证明区块链的，目的是在启动后切换到权益证明。该研究产生了结合Casper和GHOST的共识机制，发布在[Gasper论文](https://arxiv.org/abs/2003.03052)中。

根据其设计，用 python 编写了规范。 [Pyspec](https://github.com/ethereum/consensus-specs) 是一个可执行规范，可供共识层开发人员参考。它还用作客户端实现和为客户端创建测试用例向量的参考。
