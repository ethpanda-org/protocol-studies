# 以太坊协议研究员计划 (EPF)

以太坊协议研究员计划 (EPF) 是一个旨在为开发人员和研究人员提供 
直接贡献以太坊核心协议的结构化环境。该奖学金为参与者提供 
指导、学习机会和一些最紧迫技术方面的实践经验 
以太坊中的挑战。

## 为什么是 EPF？

以太坊的核心协议开发是一个高度专业化且快速发展的领域。 EPF 帮助登机 
并通过提供以下内容来支持下一代协议贡献者：

- **访问核心开发社区** – 与以太坊的顶尖研究人员和工程师一起工作。
- **真实世界的经验** – 为以太坊的执行层和共识层中的未决问题做出贡献。
- **选择您的项目** – 没有固定的课程，只有深入的技术参与。
- **无需许可的参与** – 大门是敞开的。
- **资金支持** – 一些研究员获得津贴以专注于该计划。

## 如何参与

参与的最佳方式是开始为以太坊的协议讨论和代码库做出贡献。请按照下列步骤操作：

1. **加入以太坊 R&D Discord** – 与社区互动并提出问题。 [Discord 邀请](https://discord.gg/ethereum)
2. **关注以太坊改进提案 (EIP)** – 阅读并讨论 [EthMagicians](https://ethereum-magicians.org/) 和 [ethresear.ch](https://ethresear.ch/) 上的活跃提案。
3. **探索 Good First Issues** – 通过 [Good First Issues](https://gfi.bordel.wtf) 做出您的第一个贡献。
4. **申请下一个 EPF 队列** – 请留意 [以太坊基金会博客](https://blog.ethereum.org) 上的公告。

EPF 不仅仅是编写代码，而是成为以太坊长期发展的核心部分。如果您热衷于推进以太坊的协议，这是最好的入门方式。
