# **欢迎来到协议！**

EPF Wiki 是有关 Ethereum 协议的技术资源的集合。它是 EPF 学习小组和 Protocol Wiki 的所有资源的所在地。其中包括涵盖所有核心协议领域和正在进行的研究的技术信息。它致力于为有抱负的核心贡献者提供服务，不涵盖任何智能合约或其他高级开发。对于那些希望更深入地研究并直接使用 Ethereum 协议的人来说，这也可以作为 Protocol Fellowship 的介绍。  

## Protocol Study Group

Ethereum Protocol Study Group (EPS) 是一个开放的学习计划，适合任何有兴趣了解协议内部结构的人。 学习小组资源可作为自学课程提供，但每年 2 月至 4 月期间还组织现场学习小组。关注 blog.ethereum.org，了解加入学生社区的公告。 

**学习小组 2026** 将于 2 月 23 日开始，为期 8 周的内容。与我们一起学习 [study.epf.wiki](https://study.epf.wiki)

> 学习小组不仅仅是一个资源列表，而且是一个由维护资源和学习核心协议的人员组成的社区。 **加入我们的 [Discord 服务器](https://discord.gg/8RPnPGEQtJ) 社区**。

## Protocol Wiki

Protocol Wiki 是 Ethereum 核心开发研究主要部分的技术文档。它指导任何有兴趣了解协议底层并做出贡献的人。该 wiki 由围绕学习小组形成的社区维护，并持续关注最新研究。 

> :warning: Protocol Wiki 仍在建设中，许多重要主题尚未完全涵盖。 **如果您发现缺失内容或问题，请随时 [贡献](contributing.md)**。
>
>

## Protocol Fellowship

[Ethereum Protocol Fellowship](/wiki/epf) (EPF) 是一个为对该协议深感兴趣的开发人员和研究人员提供的程序。这是一个直接与客户端团队合作并通过构建更大范围的项目做出贡献的机会。该项目为期 5 个月，通常每年 6 月至 11 月举办。请密切关注申请公告。
